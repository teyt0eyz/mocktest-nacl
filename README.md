# NACL Mock Test

เว็บฝึกทำข้อสอบจำลอง **NACL รอบทฤษฎี** — static site ล้วน ไม่ต้อง build ไม่ต้องมี backend
วางบน GitHub Pages ได้ทันที

## ฟีเจอร์

| | |
|---|---|
| **3 โหมด** | ทำเต็มชุด (1–5) · สุ่มคละทุกชุด (ตัดข้อซ้ำ) · ฝึกรายหมวด |
| **จับเวลา** | นับถอยหลัง 60/90/120/180 นาที หรือไม่จำกัด (นับขึ้น) พร้อมส่งอัตโนมัติเมื่อหมดเวลา |
| **ผังข้อสอบ** | กระโดดข้ามข้อได้ แยกกลุ่มตามหมวด แสดงสถานะตอบแล้ว/ทำเครื่องหมาย/ยังไม่ตอบ |
| **บันทึกอัตโนมัติ** | ทำค้างไว้แล้วปิดเบราว์เซอร์ได้ กลับมาแล้วกด “ทำต่อ” |
| **สรุปผล** | คะแนนรวม + คะแนนรายหมวดเรียงจากอ่อนสุด + เฉลยพร้อมคำอธิบาย |
| **ประวัติ** | เก็บ 60 ครั้งล่าสุดใน localStorage ดูสถิติดีที่สุด/เฉลี่ย |
| **อื่น ๆ** | โหมดมืด · คีย์ลัด `1`–`5` `←` `→` `F` · รองรับมือถือ · โหมดเฉลยทันที |

## เกณฑ์การให้คะแนน

| ประเภท | ข้อ/ชุด | คะแนน/ข้อ | เกณฑ์ |
|---|---|---|---|
| ปรนัย 5 ตัวเลือก (`mc`) | 115 | 2 | ถูก +2 · **ผิด −1** · ไม่ตอบ 0 (ตาม blueprint) |
| เลือกหลายคำตอบ (`ms`) | 10 | 5 | ค่าเริ่มต้น all-or-nothing (สลับเป็นให้คะแนนบางส่วนได้) |
| เขียนตอบ (`written`) | 5 | 5 | เทียบแบบไม่สนตัวพิมพ์เล็ก/ใหญ่ และตัดช่องว่าง |

รวม **130 ข้อ 305 คะแนน** ต่อชุด — ตรงกับ `points_total` ของทั้ง 8 หมวดในทุกชุด

> เกณฑ์ของข้อ `ms` และ `written` ไม่ได้ระบุไว้ใน blueprint ต้นทาง ค่าที่ใช้เป็นการประมาณให้ยอดรวมลงตัว 305 พอดี
> หากได้ข้อมูลจริงเพิ่มเติม แก้ค่าคงที่ `POINTS` ใน [assets/js/app.js](assets/js/app.js) ได้เลย

## รันบนเครื่อง

ต้องเสิร์ฟผ่าน HTTP เพราะเว็บโหลด `nacl_mocktest_data.json` ด้วย `fetch()`
(เปิดไฟล์ตรง ๆ ผ่าน `file://` เบราว์เซอร์จะบล็อก)

```bash
python3 -m http.server 8000
# เปิด http://localhost:8000
```

## Deploy ขึ้น GitHub Pages

```bash
git init
git add .
git commit -m "NACL mock test site"
git branch -M main
git remote add origin https://github.com/<username>/<repo>.git
git push -u origin main
```

จากนั้นที่ repo → **Settings → Pages** → Source: `Deploy from a branch`, Branch: `main` / `(root)` → Save
เว็บจะอยู่ที่ `https://<username>.github.io/<repo>/`

ไฟล์ `.nojekyll` มีไว้กันไม่ให้ Jekyll ไปประมวลผลโฟลเดอร์ `assets/`

## โครงสร้างไฟล์

```
index.html                 โครง UI ทุกหน้า (home / exam / result / history)
assets/css/style.css       design system สีฟ้า-ขาว + โหมดมืด + responsive
assets/js/app.js           สร้างชุดข้อสอบ · ตรวจคะแนน · timer · localStorage
nacl_mocktest_data.json    คลังข้อสอบ 5 ชุด (data source หลัก)
README_for_website_AI.md   สเปกต้นทางของชุดข้อมูล
```

## ข้อควรทราบ

- ข้อสอบ **จำลองรูปแบบและสัดส่วน** ตาม [blueprint จริง](https://nextpath.ce-nacl.com/blueprint) ไม่ใช่ข้อสอบจริง
- หมวด Cloud Computing และความสนใจทั่วไปด้านเทคโนโลยี มีคลังคำถามจำกัด จึงพบข้อซ้ำข้ามชุดได้
  โหมด “สุ่มคละ” และ “ฝึกรายหมวด” ตัดข้อซ้ำออกให้แล้ว
- คำถาม ~63% ยังไม่มี `explanation` — ถ้าเติมลง JSON เว็บจะแสดงในหน้าเฉลยให้อัตโนมัติ
- ข้อมูลทั้งหมดเก็บใน localStorage ของเบราว์เซอร์ ไม่ถูกส่งออกที่ไหน
