## รายละเอียดแต่ละหมวด + เนื้อหาย่อย

### หมวด 1 — ความรู้พื้นฐานเกี่ยวกับระบบคอมพิวเตอร์
- **คะแนน:** 20 | **ปรนัย:** 10 ข้อ | **เลือกหลายคำตอบ:** 0 ข้อ | **เขียนตอบ:** 0 ข้อ
- **เนื้อหาย่อย:**
  1. พื้นฐานระบบดิจิทัล
  2. ฮาร์ดแวร์ของคอมพิวเตอร์
  3. พื้นฐาน Cyber Security
  4. ระบบปฏิบัติการ (Operating System)
  5. Input/Output Loop ในระบบคอมพิวเตอร์
  6. สถาปัตยกรรมคอมพิวเตอร์ (Instruction Set Architecture)
  7. ลำดับชั้นของหน่วยความจำ (Memory Hierarchy)

### หมวด 2 — ระบบจำนวน, ระบบเลขฐาน และการดำเนินการทางคณิตศาสตร์ในคอมพิวเตอร์
- **คะแนน:** 10 | **ปรนัย:** 5 ข้อ | **เลือกหลายคำตอบ:** 0 ข้อ | **เขียนตอบ:** 0 ข้อ
- **เนื้อหาย่อย:**
  1. การดำเนินการทางคณิตศาสตร์ของเลขฐานสอง
  2. การแปลงเลขฐานสอง, ฐานสิบ, ฐานสิบหก
  3. การดำเนินการระดับบิต (Bitwise Operation) ได้แก่ AND, OR, XOR

### หมวด 3 — พื้นฐานการเขียนโปรแกรมภาษาซี
- **คะแนน:** 15 | **ปรนัย:** 5 ข้อ | **เลือกหลายคำตอบ:** 1 ข้อ | **เขียนตอบ:** 0 ข้อ
- **เนื้อหาย่อย:**
  1. Data type
  2. การวิเคราะห์การทำงานของฟังก์ชัน
  3. Pointer และการใช้งาน Pointer

### หมวด 4 — การประยุกต์ใช้คณิตศาสตร์ในระบบคอมพิวเตอร์
- **คะแนน:** 20 | **ปรนัย:** 10 ข้อ | **เลือกหลายคำตอบ:** 0 ข้อ | **เขียนตอบ:** 0 ข้อ
- **เนื้อหาย่อย:**
  1. การคำนวนผู้ใช้งานของระบบ (Concurrent users)
  2. คณิตศาสตร์การเข้ารหัส
  3. ความน่าจะเป็นเบื้องต้น
  4. ทฤษฎีการคำนวน (Theory of Computation)
  5. การวิเคราะห์องค์ประกอบหลัก (Principal Component Analysis)

### หมวด 5 — ความรู้ด้านระบบเครือข่ายคอมพิวเตอร์ (Computer Networks)
- **คะแนน:** 85 | **ปรนัย:** 25 ข้อ | **เลือกหลายคำตอบ:** 5 ข้อ | **เขียนตอบ:** 2 ข้อ
- **เนื้อหาย่อย:**
  1. แนวคิดและแบบจำลอง Client-Server Model
  2. อุปกรณ์ Network Devices และ Link ในระบบเครือข่าย
  3. ระบบเครือข่ายไร้สาย (Wireless Network)
  4. การใช้งาน Public / Private IP Address
  5. การคำนวณและการแบ่ง Subnet
  6. การทำงานของ Network Address Translation (NAT)
  7. Routing Protocol
  8. โปรโตคอลในชั้น Application Layer Protocol
  9. การใช้งาน HTTP Method และ HTTP Status Code
  10. การวัดและประเมินประสิทธิภาพของระบบเครือข่าย (Network Performance)

### หมวด 6 — ความรู้ด้านการประมวลผลแบบคลาวด์ (Cloud Computing)
- **คะแนน:** 70 | **ปรนัย:** 25 ข้อ | **เลือกหลายคำตอบ:** 2 ข้อ | **เขียนตอบ:** 2 ข้อ
- **เนื้อหาย่อย:**
  1. เทคโนโลยี Virtualization
  2. Load balance
  3. บริการ Public Cloud ต่าง ๆ (ครอบคลุมเฉพาะ GCP / Azure / AWS)
  4. การดูแลระบบ Linux
  5. เทคโนโลยี Container
  6. Infrastructure as Code (IaC)
  7. System Design บนระบบ Cloud

### หมวด 7 — ความสนใจโดยรวมด้านเทคโนโลยี / วิศวกรรมคอมพิวเตอร์ / Computer Network / Cloud Computing
- **คะแนน:** 50 | **ปรนัย:** 20 ข้อ | **เลือกหลายคำตอบ:** 1 ข้อ | **เขียนตอบ:** 1 ข้อ
- **เนื้อหาย่อย:** _ไม่ระบุรายการเนื้อหาย่อยบนหน้า blueprint_
